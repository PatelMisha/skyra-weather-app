# Skyra Weather App

A full-featured weather dashboard built with Java + JavaFX.

## Features
- 🔍 Live city autocomplete search
- 🌡 Real-time weather data (OpenWeatherMap API)
- ⏱ 24-hour hourly forecast with temperature chart
- 📅 7-day forecast calendar
- ☀️ Auto day/night theme based on city's local time
- 🕐 Live local clock for searched city
- 💻 Responsive desktop layout

## Run
```bash
mvn clean compile javafx:run
```
